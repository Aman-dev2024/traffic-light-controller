# traffic-light-controller
Traffic light controller is a small game in which you can change the traffic light by clicking on button.

✨ Features
. Change light to red
. Change light to yellow
. Change light to green
. Buttons to Change Light

🛠️ Tech Stack
. HTML5
. CSS3  
. JS 

📖 How to Use
1) Open the file index.html in your browser.
2) Click on the button of which color of signal you want.
🔗 Live demo: https://aman-dev2024.github.io/traffic-light-controller/
